                # 安装 Docker Desktop

                # 1 Windows 安装 Docker

                # 1.1 下载并安装 Docker

                官方文档：在 Windows | 上安装 Docker DesktopDocker 文档open in new window

                系统要求：

                查看系统版本：win+r 输入winver

                注意最后一行需要安装 WSL2 Linux 内核更新包，否则运行Docker Desktop会遇到以下错误

                # 1.3 配置阿里云镜像加速

                阿里云镜像加速器地址：参见：容器镜像服务 (aliyun.com)open in new window

                在 Settings --> Docker Engine 中配置镜像加速

                {
  "registry-mirrors": ["https://7ub9273f.mirror.aliyuncs.com"]
}
